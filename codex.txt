SORUN 3 — YAZILAR / ARTICLES SAYFASI HATALI:
Yazılar sayfasında yazı kartları görünüyor ama:
- Yazı kartına tıklayınca detay sayfasına gidilmiyor.
- Yazı detay sayfası yoksa oluşturulmalı.
- Bazı yazılarda görsel alanı kırık/boş görünüyor.
- Görsel yoksa kırık img veya alt text gibi görünen bozuk alan oluşmamalı.
- Kartın tamamı tıklanabilir olmalı.

Yap:
- /articles sayfasındaki her yazı kartı tamamen tıklanabilir olsun.
- Link hedefi: /articles/{slug}
- /articles/[slug] route’u yoksa oluştur.
- Detay sayfasında yazı gösterilsin.
- Slug yoksa başlıktan güvenli slug üret.
- Slug çakışmasını engelle.
- Slug boş olamaz.
- Yayında olmayan yazılar public /articles sayfasında görünmesin.
- Yayında olmayan yazı detayına gidilirse 404 dönsün.
- Bulunamayan slug 404 dönsün.
- Yazı kartlarında kırık img gösterme.
- imageUrl boşsa veya geçersizse görsel alanını gizle veya düzgün placeholder göster.
- Görsel alanı kartı aşırı büyütmesin.
- Görsel varsa aspect-video veya h-48 gibi kontrollü oran kullan.
- Görsel varsa object-cover kullan.
- Görsel alt text yazı başlığından gelsin.
- Görsel yoksa kırık ikon/alt text görüntüsü oluşmasın.
- Kart hover/focus düzgün olsun.
- Keyboard erişilebilirliği olsun.
- Mobilde tek kolon, desktopta düzgün grid olsun.
- Kart içerikleri taşmasın.

Yazı detay sayfasında gösterilecek alanlar:
- başlık
- tarih
- kategori
- etiketler
- okuma süresi
- kapak görseli varsa
- içerik
- geri dön linki

SEO:
- /articles/[slug] için metadata ekle:
  - title
  - description
  - openGraph
  - twitter

Admin yazılar tarafında:
- Yazı ekleme/düzenleme sırasında imageUrl doğru kaydediliyor mu kontrol et.
- Upload sonrası dönen URL article imageUrl alanına doğru yazılıyor mu kontrol et.
- imageUrl formatı /uploads/dosyaadi.ext olmalı.
- Harici URL destekleniyorsa kırık görüntüye sebep olmasın.
- Slug alanı boşsa başlıktan otomatik üret.
- Başlık değişince mevcut slug gereksiz bozulmasın.
- published default true veya mevcut mantığa uygun olsun.
- tags array olsun.
- readingTime tutarlı olsun.
- date YYYY-MM-DD veya ISO olarak tutarlı olsun.
